# customelements

Setup minimale per un progetto web con Parcel, Prettier e Biome.

## Script npm
- `npm run dev` avvia il server di sviluppo Parcel
- `npm run build` crea la build di produzione
- `npm run clean` pulisce la cache e la cartella dist
- `npm run lint` esegue il linter Biome sui file JS
- `npm run format` formatta HTML/CSS/JSON con Prettier e JS con Biome

## Struttura
- `src/index.html` entry HTML
- `src/main.js` entry JS
- `src/styles.css` stili di base
